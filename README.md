Script untuk explorer (Severless)


1. Buat Worker di cloudflare
2. Upload script ke worker via wrangler / Manual di web cloudflare
3. Ambil RPC Crypto (Cari yang free dan berbayar untuk backup)
4. Buat Variable secret di cloudflare MY_RPC_URL dan MY_RPC_URL2 untuk RPC berbayar
5. Untuk scraping data lainnya kita pakai yang free dari blockscout
6. daftar blockscout lalu ambil API nya https://base.blockscout.com/account
7. Lalu buat juga variable secret di cloudflare untuk BLOCKSCOUT_API_KEY


Demo : https://baseplorer.iqbal.sbs/
